# AlgoProphet Frontend

This is the institutional frontend for the AlgoProphet decentralized prediction market.

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** (Institutional dark theme)
- **JetBrains Mono** (Monospace font)
- **Lucide React** (Icons)
- **Framer Motion** (Animations)

## Development

```bash
npm install
npm run dev
```

Runs at `http://localhost:3000`

# AlgoProphet: Institutional AI-Native Prediction Markets

"Bridging the Gap Between Autonomous Intelligence and Decentralized Commerce."

AlgoProphet is a high-performance, decentralized prediction market platform built on the Algorand Blockchain. The platform features Aegis, a sovereign AI trading agent that autonomously analyzes market sentiment and executes real-time trades using a custom "Cautious Bull" strategy.

---

## Core Architecture

AlgoProphet is designed for the next generation of Agentic Commerce, where AI agents serve as first-class citizens in the DeFi ecosystem.

- Aegis Intelligence Loop: An autonomous decision engine (App ID 1005) that calculates sentiment vectors and executes trades in 3-tier risk tranches: Aggressive, Moderate, and Defensive.
- Institutional "Obsidian" Interface: A brutalist, high-contrast dashboard optimized for professional oversight of AI-driven market activity.
- Multi-View Navigation: A sophisticated client-side routing system providing dedicated views for Live Markets, Active Portfolio (My Bets), and a Terminal-Style Transaction History.

---

## Key Features

- Pera Wallet Handshake: Full integration with the Pera Connect SDK, allowing for secure QR-code-based account linking and session persistence.
- Real-Time Oracle Canvas: A dynamic UI featuring live probability drifting, SVG price charts, and a streaming Intelligence Feed of agent thoughts.
- One-Click Settlement: An integrated Winning Claims system that allows users to realize profits and settle positions directly from the Portfolio view.
- Visual Event Indexing: Custom Cyan Shadow Pulse animations triggered by live blockchain events via the Algorand Indexer.

## Getting Started

### Prerequisites

- Algorand Localnet (via AlgoKit)
- Node.js v22.x

### Setup

1. Clone the Repository
    
    git clone https://github.com/marrihemanth/AlgoProphet-Decentralized-Prediction-Market.git
    
2. Install Dependencies
    
    npm install
    
3. Environment Configuration
    
    Populate .env.local with your deployed App IDs (Market: 1004, Orderbook: 1005).
    
4. Launch the System

Terminal 1: Frontend

npm run dev

Terminal 2: Aegis Agent

npx tsx projects/algo-prophet-agent/scripts/aegis_loop.ts
